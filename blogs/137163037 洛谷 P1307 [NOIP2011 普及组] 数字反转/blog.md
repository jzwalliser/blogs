# 题目
洛谷 [P1307 [NOIP2011 普及组] 数字反转](https://www.luogu.com.cn/problem/P1307#submit)
> ## [NOIP2011 普及组] 数字反转
>
> ### 题目描述
>
> 给定一个整数 $N$，请将该数各个位上数字反转得到一个新数。新数也应满足整数的常见形式，即除非给定的原数为零，否则反转后得到的新数的最高位数字不应为零（参见样例 2）。
>
> ### 输入格式
>
> 一个整数 $N$。
>
> ### 输出格式
>
> 一个整数，表示反转后的新数。
>
> ### 样例 #1
>
> #### 样例输入 #1
>
> ```
> 123
> ```
>
> #### 样例输出 #1
>
> ```
> 321
> ```
>
> ### 样例 #2
>
> #### 样例输入 #2
>
> ```
> -380
> ```
>
> #### 样例输出 #2
>
> ```
> -83
> ```
>
> ### 提示
>
> **【数据范围】**
>
> $-1,000,000,000\leq N\leq 1,000,000,000$。
>
> noip2011 普及组第一题
> 

# 想法
先注意一下题目范围：$-1,000,000,000\leq N\leq 1,000,000,000$，所以肯定是要用`long long int`的，否则绝对存不下。
之后考虑一些特殊情况：如果输入是0，那就直接输出0。
然后，分类讨论正数和负数。若输入的是负数，那么就把它转化为正数后再处理，不过到时候输出的时候就得先输出一个负号`-`。
最后，得到纯净的数据后，就要考虑如何把数字拆开了倒序输出了。我的办法是将数字模10获得最低位，之后再把数字除10把最低为去掉，循环往复以上操作直到数字变成0。
也要注意，不能输出前导0。这个也好办，可以使用一个开关，默认`false`。在做除法的时候，碰到0就检查开关状态，如果发现是`false`的那就不输出0。直到碰到非0的数后，将开关设为`true`，这样就不会一并过滤掉需要输出的0了。

# 实现
1. 输入
2. 检查输入是否为0。
3. 如果是负数（即输入小于0）那就把它转化为正数，同时输出一个负号。
4. 一位位地处理数字，碰到0检查开关状态以决定是否输出它。

# 题解
## C++
```cpp
#include<bits/stdc++.h>
using namespace std;
long long int num;
int main(){
    cin >> num; //输入
    if(num == 0){ //特判0
        cout << 0; //直接输出0
        return 0; //结束程序
    }
    if(num < 0){ //负数
        cout << '-'; //输出负号
        num = -num; //转化为正数
    }
    bool flag = false; //开关初始是false
    while(num){ //执行一下步骤直到数字变成0
        if(num % 10 == 0 and flag == false){} //如果是前导零就不输出，这个if段虽然什么都不做但不可以去掉，因为它毙掉了一个情况
        else{ //输出
            flag = true;
            cout << num % 10;
        }
        num /= 10; //使数字的倒数第二位露出来
    }
    return 0;
}
```

## Python
```py
num = int(input()) #输入
if num == 0: #特判0
	print(0) #直接输出0
	raise SystemExit #结束程序
if num < 0: #负数
	print('-',end='') #输出负号
	num = -num #转化为正数

flag = False #开关初始是False
while num: #执行一下步骤直到数字变成0
	if num % 10 == 0 and flag == False: #如果是前导零就不输出，这个if段虽然什么都不做但不可以去掉，因为它毙掉了一个情况
		pass
	else: #输出
		flag = True
		print(num % 10,end = '')
	num //= 10 #使数字的倒数第二位露出来
```
