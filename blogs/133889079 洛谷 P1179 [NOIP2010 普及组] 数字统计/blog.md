# 题目
洛谷 [P1179 [NOIP2010 普及组] 数字统计](https://www.luogu.com.cn/problem/P1179)
>## [NOIP2010 普及组] 数字统计
>
>### 题目描述
>
>请统计某个给定范围 $[L, R]$ 的所有整数中，数字 $2$ 出现的次数。
>
>比如给定范围 $[2, 22]$，数字 $2$ 在数 $2$ 中出现了 $1$ 次，在数 $12$ 中出现 $1$ 次，在数 $20$ 中出现 $1$ 次，在数 $21$ 中出现 $1$ 次，在数 $22$ 中出现 $2$ 次，所以数字 $2$ 在该范围内一共出现了 $6$ 次。
>
>### 输入格式
>
>$2$ 个正整数 $L$ 和 $R$，之间用一个空格隔开。
>
>### 输出格式
>
>数字 $2$ 出现的次数。
>
>### 样例 #1
>
>#### 样例输入 #1
>
>```
>2 22
>```
>
>#### 样例输出 #1
>
>```
>6
>```
>
>### 样例 #2
>
>#### 样例输入 #2
>
>```
>2 100
>```
>
>#### 样例输出 #2
>
>```
>20
>```
>
>### 提示
>
>$1 ≤ L ≤R≤ 100000$。
# 想法
题目意思已经很明确了。这里，我们可以将每一个数字一位一位拆开来，然后看数字中到底有几个$2$。最后，再将统计到的$2$全部加起来即可。注意范围中包括$L$和$R$。
再看数据规模，也就$1 ≤ L ≤R≤ 100000$，其实不算大，暴力一下很容易就通过了。
# 实现
1. 写一个函数，用于统计一个数字中$2$出现的个数。
2. 从$L$枚举到$R$，然后统计数字$2$。
3. 最后输出。
# 题解
## C++

```cpp
#include<bits/stdc++.h>
using namespace std;

int deal(int num){ //函数，用于统计一个数字中2出现的次数
	int two = 0;
	while(num != 0){
		if(num % 10 == 2){
			two += 1;
		}
		num /= 10;
	}
	return two;
}

int main(){
	int l,r;
	int two = 0; //统计到的2点个数
	cin >> l >> r; //输入
	for(int i = l;i <= r;i++){
		two += deal(i); //处理每一个数字
	}
	cout << two;
}
```
## Python
```py
def deal(num): #函数，用于统计一个数字中2出现的次数
	two = 0
	while num != 0:
		if num % 10 == 2:
			two += 1
		num = int(num / 10)
	return two

two = 0 #统计到的2点个数
get = input().split() #输入
l = int(get[0])
r = int(get[1])

for i in range(l,r+1):
	two += deal(i) #处理每一个数字

print(two)
```
