# 三原色
三原色一共有两类：分别是颜料三原色和色光三原色。

传统美术中的色调三原色包括红、蓝、黄：用这三种颜料可以拼出几乎任何一种颜料（不过CMYK系列的混色范围更广）；而色光三原色则是红、绿、蓝。在光学中，这三种颜色可以通过不同比例混合，拼出几乎任何一种色光。

![请添加图片描述](https://i-blog.csdnimg.cn/direct/e148201f8c334f44bd314ac8da42891d.png)


而光学三原色是RGB色系的基础，也是本篇的主角：因为你的电脑屏幕是发光的，而不是通过到处喷洒颜料来显示的。

在普通的消费级屏幕上，屏幕上任何一种颜色都可以用RGB表示出来。这也就是为什么很多屏幕测试工具都会让屏幕显示出红、绿、蓝三种颜色。

有趣的是，如果你把手机或电脑屏幕放在放大镜或微距镜头下观察，你会发现屏幕上到处都是密密麻麻的红、绿、蓝三色小点——它们正是发光的像素。还有更简单的方法：在屏幕上放一滴水，也能观察到类似现象。

# RGB
RGB的底层原理就是“调色”：通过调整像素上红、绿、蓝三种颜色的“含量”，可以显示出不同的颜色。
![请添加图片描述](https://i-blog.csdnimg.cn/direct/a3627b9f94eb4c359bbf23516c26d334.png)


RGB模型包含三个数值，一般来说用一个三元组（如(19,198,10)）或三维列表来表示。这三个数值的顺序是固定的，从左到右分别表示红（Red）、绿（Green）、蓝（Blue）的含量，如下图。每个数值都介于0～255（闭区间）之间，其中0代表不存在该颜色，255代表将这个颜色的发光强度最大。

![请添加图片描述](https://i-blog.csdnimg.cn/direct/7a15ea57bffd4085884ecdfe07b25a50.png)

由于三个通道的“含量”都被分成了256份，因而RGB能表示出共$256\times256\times256=16777216=2^{24}$种不同的颜色。

# HEX格式的RGB
HEX不是一个新的色系，而是RGB颜色的一种常见的十六进制记法。这种颜色长这样：#ABCDEF。例如：

具体规则是：去掉开头的 `#`，把剩下的字符串每两个字符切为一段。第1-2个字符（十六进制）表示红色，第3-4个字符表示绿色，第5-6个表示蓝色。

# RGBA
有的时候，你会见到“RGBA色系”。其实它和RGB色系是一样的，只是在后者的基础上，加了个透明度（Alpha）通道。透明度介于0到1之间，其中1代表不透明，0代表完全透明。
