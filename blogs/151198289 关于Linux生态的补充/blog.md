# Linux之生态

Look！这就是我电脑上运行的Zorin OS。通过软件包管理器，可以发现其实**源基本都和Ubuntu脱不了干系**。全是“ubuntu jammy”……
![请添加图片描述](https://i-blog.csdnimg.cn/direct/c139b49f1c424aa6b79d92e92893ef38.png)

甚至用apt安装软件的时候，都能发现是从清华源下载的Ubuntu jammy的软件包。
![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/2638d2d539994963ac1859a4968e2890.png#pic_center)

其次，我们会发现很多网站上只提供几种格式的Linux包，就以NTQQ为例，官网上只有deb格式、rpm格式和appimage的包。如果真如那位键盘侠所说，Linux兼容性非常差，那我相信NTQQ应该会有Ubuntu版、Debian版、Kubuntu版、Lubuntu版、Manjaro版、Redhat版、openSUSE版、Arch版、Alpine版、Gentoo版、Zorin OS版等等等等。那为什么不是这样呢？说明 Linux兼容性是可以的，一个deb包就能解决几乎所有Debian系列的发行版，rpm能几乎解决所有Fedora系列的发行版，剩下的就用Appimage带过。

而且，就连**Zorin OS官网上，也说这款操作系统兼容所有Ubuntu的软件**。
![请添加图片描述](https://i-blog.csdnimg.cn/direct/baf7f9d215c34daa8d61a5bbd6054674.png)
> Zorin OS is also natively compatible with third-party Ubuntu & Linux apps.

当然我也承认Linux在兼容方面确实有待进步，但是**绝对没有那么差劲**。
# 思考
Ok，列举完理由，我们再看看那位网友所说。
针对我说的“Zorin OS是基于Ubuntu的，这意味着他有不错的生态”，他表示：

>现在看到XXX基于XXX所以软件都有就想笑，就Linux那兼容性，基于XXX就能用XXX的软件了？想Peach！

在我看来：

1）他对Linux**没有一个充分的认识**，然而表达欲非常强烈。

2）他**没有亲自试验过Zorin OS的软件兼容性**，就凭着或许并不丰富的经验认为我说的是错的。

3）他以一种极为**幼稚**的表达方式体现出对于我的回答的不满，而**语言本身也非常失礼**。

4）他学会了阴阳的艺术，然而**在网络上使用却非常不合适**。

若他能够以更加温和有好的语气和方式说出他的想法，即使他的想法与我的固有认知不一样，我依然会给他点个赞。然而以此等恶劣的态度加傲慢无礼的语言发布这么一条不伦不类的评论，额，对不起，我的评论区内不欢迎这样的人。

所以，举报+删除。
