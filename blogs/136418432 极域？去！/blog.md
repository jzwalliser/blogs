# 方法（操作）
在介绍“小妙招”之前，请注意：一会儿的操作是有副作用的。**你将无法看到老师上课的演示**（当然你可以偷看同桌的屏幕，并让同桌羡慕你居然逃离了老师的控制），**老师给你发送的文件你也将无法收到**。

## 方法一
最简单粗暴的方法，莫过于拔网线了。找到电脑的网线接口，然后直接拔掉即可。

|缺点|
|:--|
|1. 网线难找，容易拔错线|
|2. 操作困难，网线一般都在犄角旮旯里，很难拔出来|
|3. 网线很难插回去|


|优点|
|:--|
|1. 技术难度低|
|2. 网线插回去后，投屏、文件发送可立即恢复|

|副作用|
|:--|
|1. 无法接受投屏|
|2. 无法接受文件|
|3. 可能无法上网|
|4. 教师端提示下线|
|5. 可能会被老师骂|

## 方法二
在执行本方法前，建议搞清楚极域的可执行文件到底在哪里。否则，想重新打开极域可能会比较困难。

众所周知，用任务管理器是无法结束进程`studentmain.exe`（即，极域）的，会“拒绝访问”（Windows 7好像可以偶尔杀掉进程，但感觉Windows 10不行）。所以，使用`taskkill`命令：
打开cmd，输入：
```
taskkill /F /IM studentmain.exe
```
这样就可以把极域的后台进程杀掉。不过和拔网线一样，教师端会显示你掉线了~

|缺点|
|:--|
|1. 技术难度高（可能很多同学都不会用cmd，甚至不知道它是什么）|
|2. 极域的快捷方式不一定很容易找到（我们后面会提到）|

|优点|
|:--|
|1. 方便|
|2. 成功率极高|


|副作用|
|:--|
|1. 无法接收投屏|
|2. 无法接收文件|
|3. 教师端提示下线|
|4. 可能会被老师骂|





### Tip
若不想每次上课都敲一遍命令，也可以将命令保存到bat或cmd文件中，并存到你的u盘。这样每次双击打开可执行文件（即bat/cmd文件）即可。不过如果有U盘锁，那就算了吧……



# 成功绕过了极域，但想重新获得老师的投屏/文件

如果你通过拔网线的方式获得了自由，那么把网线重新插回去，极域会自动连接。

若你参照方法二，把极域的进程给杀了，那重新启动极域就是啦。一般来说，极域都会有一个快捷方式在桌面上，双击即可。

但有时，桌面上并没有极域的快捷方式（如nwhx机房中的电脑）。此时可以在`C:\Program Files (x86)\Mythware\极域课堂管理系统软件v6.x`中看看有没有`studentmain.exe`。不过这种操作一定很烦吧，所以最好还是在杀`studentmain.exe`之前，通过任务管理器了解`studentmain`在哪里，提前记录下来备用啊。后面将讲解方法。

# 屏幕锁定过程中如何突破
如果你不幸被老师控制了屏幕，无法输入命令杀死极域，那有挽救方法吗？
答：有。

## Windows 7
对于windows 7，需要你手速快，须在几秒内完成以下步骤：

1. 按下<kbd>Win</kbd> + <kbd>L</kbd>锁定账户
2. 按下<kbd>Alt</kbd> + <kbd>W</kbd>切换账户
3. 找到你刚才的账户，重新登录，若找不到刚才的账户，则按下<kbd>Esc</kbd>即可，这样就可找到
4. 这时候你会有很短暂的喘息时间，趁着这短暂的几秒，赶紧打开cmd运行命令：
```
taskkill /F /IM studentmain.exe
```

5. 如果尚未输完命令极域就反应过来，把你屏幕锁了，那就重新从第一步开始，短暂解锁电脑屏幕后，接着输入，输完后回车

## Windows 10
如果键盘上有睡眠键，那就太棒了。否则，就很可能没办法了（如果你有办法可以评论区回复哦🎉）……
1. 按下睡眠<kbd>🌙</kbd>键
2. 等待电脑睡眠后，将电脑唤醒
3. 这时候你会有很短暂的喘息时间，趁着这短暂的几秒，赶紧打开cmd运行命令：
```
taskkill /F /IM studentmain.exe
```
4. 如果尚未输完命令极域就反应过来，把你屏幕锁了，那就重新从第一步开始，短暂解锁电脑屏幕后，接着输入，输完后回车

# U盘锁
U盘插入电脑，打开却显示“拒绝访问”。怎么办？
打开CMD，输入：
```
sc stop tdfilefilter
```
回车，U盘应该就可以正常使用了。
