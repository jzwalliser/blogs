# 题目
洛谷 [P2010 [NOIP2016 普及组] 回文日期](https://www.luogu.com.cn/problem/P2010)

>## [NOIP2016 普及组] 回文日期
>### 题目背景
>NOIP2016 普及组 T2
>### 题目描述
>在日常生活中，通过年、月、日这三个要素可以表示出一个唯一确定的日期。
>牛牛习惯用 $8$ 位数字表示一个日期，其中，前 $4$ 位代表年份，接下来 $2$ 位代表月份，最后 $2$ 位代表日期。显然：一个日期只有一种表示方法，而两个不同的日期的表 示方法不会相同。
>牛牛认为，一个日期是回文的，当且仅当表示这个日期的 $8$ 位数字是回文的。现在，牛牛想知道：在他指定的两个日期之间包含这两个日期本身），有多少个真实存在的日期是回文的。
>一个 $8$ 位数字是回文的，当且仅当对于所有的 $i$（$1  \le i \le 8$）从左向右数的第 $i$ 个数字和第 $9-i$ 个数字（即从右向左数的第 $i$ 个数字）是相同的。
>例如：
>- 对于 2016 年 11 月 19 日，用 $8$ 位数字 $20161119$ 表示，它不是回文的。
>- 对于 2010 年 1 月 2 日，用 $8$ 位数字 $20100102$ 表示，它是回文的。
>- 对于 2010 年 10 月 2 日，用 $8$ 位数字 $20101002$ 表示，它不是回文的。
>每一年中都有 $12$ 个月份：
>其中，$1, 3, 5, 7, 8, 10, 12$ 月每个月有 $31$ 天；$4, 6, 9, 11$ 月每个月有 $30$ 天；而对于 $2$ 月，闰年时有 $29$ 天，平年时有 $28$ 天。
>一个年份是闰年当且仅当它满足下列两种情况其中的一种：
>1. 这个年份是 $4$ 的整数倍，但不是 $100$ 的整数倍；
>2. 这个年份是 $400$ 的整数倍。
>例如：
>- 以下几个年份都是闰年：$2000, 2012, 2016$。
>- 以下几个年份是平年：$1900, 2011, 2014$。
>### 输入格式
>两行，每行包括一个 $8$ 位数字。
>第一行表示牛牛指定的起始日期。
>第二行表示牛牛指定的终止日期。
>保证 $\mathit{date}_1$ 和 $\mathit{date}_2$ 都是真实存在的日期，且年份部分一定为 $4$ 位数字，且首位数字不为 $0$。
>保证 $\mathit{date}_1$ 一定不晚于 $\mathit{date}_2$。
>### 输出格式
>一个整数，表示在 $\mathit{date}_1$ 和 $\mathit{date}_2$ 之间，有多少个日期是回文的。
>### 样例 #1
>#### 样例输入 #1
>```
>20110101
>20111231
>```
>#### 样例输出 #1
>```
>1
>```
>### 样例 #2
>#### 样例输入 #2
>```
>20000101
>20101231
>```
>#### 样例输出 #2
>```
>2
>```
>### 提示
>**【样例说明】**
>对于样例 1，符合条件的日期是 $20111102$。
>对于样例 2，符合条件的日期是 $20011002$ 和 $20100102$。
>**【子任务】**
>对于 $60 \%$ 的数据，满足 $\mathit{date}_1 = \mathit{date}_2$。
# 想法

这道题很显然是用枚举，但这道题的枚举肯定是要有技巧的。

有这么几种枚举法：
①枚举八位数，然后检查其是否同时为回文数和合法日期。
②枚举合法日期，检查是否为回文数。
③枚举回文数，判断是否为合法日期。

很显然算法③更优，①肯定是最差的了。而对于②来说，每年都有365天，遇到闰年还要特判一下`xxxx0229`是否回文；不仅写起来麻烦，还容易出错。

算法③则最优，因为每年最多只有一个回文日期，可以“剪”掉许多无效数据，并且无需考虑闰年。程序仅需枚举年份，然后照此构造回文数，最后检查合法性即可。如`2010`可构造出回文书`20100102`；2024可构造出`20244202`；`9220`可构造出`9200229`，但`2024`构造出的`20244202`不是合法日期。

至于如何检测是否是合法日期呢？如果日期最后两位超过了当月最大的天数，那就不是合法日期（如`23100132`），月份大于12也不合法。

# 实现
1. 输入。
2. 枚举4位数，构造8位数。
3. 判断8位数是否合法日期。
4. 输出。

# 实现

## C++

```cpp
#include<bits/stdc++.h>
using namespace std;
int days[100] = {-1,31,29,31,30,31,30,31,31,30,31,30,31}; //每个月的天数，“-1”用于站位
int main(){ 
    int a,b,count = 0;
    cin >> a >> b;
    int y = a / 10000; //年，前4位数
    while(true){
        int m = y % 10 * 10 + y % 100 / 10; //月，中间2位数
        int d = y % 1000 / 100 * 10 + y / 1000; //天，最后2位数
        int merged = y * 10000 + m * 100 + d; //合并后的日期
        if(days[m] >= d and merged >= a and merged <= b){ //是否介于开始日期和结束日期之间
            count += 1; //如果是，那么下一年
    	}
        if(merged > b){ //如果越界了，那么结束程序
            cout << count; //输出结果
            return 0; //结束程序
        }
        y += 1; //如果还没越界，那就判断下一年
    }
}

```
## Python
```py
import sys
days = [-1,31,29,31,30,31,30,31,31,30,31,30,31] + [0] * 100 #每个月的天数，“-1”用于站位
count = 0


a = int(input())
b = int(input())

y = a // 10000 #年，前4位数
while True:
    m = y % 10 * 10 + y % 100 // 10 #月，中间2位数
    d = y % 1000 // 100 * 10 + y // 1000 #天，最后2位数
    merged = y * 10000 + m * 100 + d #合并后的日期
    if days[m] >= d and a <= merged <= b: #是否介于开始日期和结束日期之间
        count += 1 #如果是，那么下一年
    if merged > b: #如果越界了，那么结束程序
        print(count) #输出结果
        sys.exit() #结束程序
        
    y += 1 #如果还没越界，那就判断下一年
```
