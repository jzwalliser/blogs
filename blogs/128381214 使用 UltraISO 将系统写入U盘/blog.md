# 前言
有的时候，我们需要安装一个系统，却只有 iso 镜像文件。怎么办呢？这时候，就需要将 iso 文件写入U盘了。你可以使用 Rufus，Ventoy，但这次我们主要来看看如何使用UltraISO。文末我会提供UltraISO的下载链接。
# 写入系统
## 1. 打开文件
在网上下载到iso镜像文件后，我们使用UltraISO将其打开。点击菜单栏上的“文件（F）”，选择“打开”，然后加载ISO文件。
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/da56f82240a4aa55455ca391261d2f20.png)
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/4f4b33d535a1077e09e47ac9119d6623.png)
## 2. 写入U盘
打开文件后，点击菜单栏中的“启动（B）”，选择“写入磁盘映像”。
![内容](https://i-blog.csdnimg.cn/blog_migrate/534f8979575d4508ec1cb4178a531d53.png)![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/2a00c03bd023a68044968f252d0fe66b.png#pic_center)在跳出的窗口中选择“写入”。记得一定要提前备份U盘中的文件！因为写入操作会格式化U盘，届时U盘中所有的文件将会被擦除。

## 3. 等待
整个写入过程就要看U盘的读写速度啦。5分钟后，总算完成啦！![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/08ff357987c38d8c585039ad4ab215bd.png#pic_center)
