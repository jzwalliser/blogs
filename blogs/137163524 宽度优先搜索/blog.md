# 宽搜

在介绍宽搜的原理之前，咱先玩个游戏：走迷宫。给定这样一个迷宫，请问怎样走？
![请添加图片描述](https://i-blog.csdnimg.cn/direct/3b89deddd90d41f3a4e8ac513b3a18ad.png)

对于人来说，答案是很显然的。但如果是台电脑需要解决这个问题，并没有你这么强大的头脑和眼力，没法一眼看出结果，那改怎么办呢？
此时，搜索算法就派上了用场。
搜索主要包括两种比较常用的算法：深搜和宽搜。这一章，主要介绍宽搜。
# 算法
回到前面那个问题：电脑怎样走迷宫呢？或者说吧，程序员是怎样设计走迷宫的程序的？
可以看一下这幅动图，或许会有所启发（制作得比较粗糙，能看懂就好啦哈哈哈）。


[video(video-XLlUYwx8-1781088631374)(type-csdn)(url-https://live.csdn.net/v/embed/530552)(image-https://v-blog.csdnimg.cn/asset/e8f1bf796178bc2cf512ec4ef204ec10/cover/Cover0.jpg)(title-)]

容易发现，程序将每一条路都走了一遍，最终走到了终点。并且，每条路都是同步走的。所以，思路是怎样的呢？
让我们以一个较为简单的迷宫为例，分析一下步骤。
```
 ABCDE
1x xxx
2x x  
3    x
4xxxxx
```
入口为A3，出口为E2，该怎么走？
首先，建立一个链表（充当队列），把入口坐标放入。


|<kbd>A3</kbd>|
|:-|

接着，把队列中的第一个元素取出，发现是A3，此时对其进行分析：从A3开始，可以往哪儿走？发现只有一个选择：只能走到B3。因此 把B3压入队列。（主意刚才的A3已被取出，因此现在队列中只剩下B3）
|<kbd>B3</kbd>|
|:-|

再做同样的操作：把队列中头部元素取出，发现是B3。B3这回有两个选择了：即可以往B2走，也可以往C3走。因此将B2、C3压入队列：
|<kbd>B2</kbd><kbd>C3</kbd>|
|:-|

把队列中第一个元素，即B2取出发现由B2可以走向B1，所以把B1放入队列：
|<kbd>C3</kbd><kbd>B1</kbd>|
|:-|

C3可以走向D3，所以：
|<kbd>B1</kbd><kbd>D3</kbd>|
|:-|

再把B1取出，发现由B1无法再往任何地方走，是死路一条。因此，由B1无法得到下一步该往哪儿走，即没有元素可以加入队列。
|<kbd>D3</kbd>|
|:-|

由D3可以走到D2。
|<kbd>D2</kbd>|
|:-|

由D2可以走到E2。
|<kbd>E2</kbd>|
|:-|

发现E2就是出口，因此程序运行结束。那么，如果需要知道入口到出口的完整路径，那么在刚才的过程中，可以为每个坐标都记录下它是由谁走来的，像这样：
```
  A  B  C  D  E
1 x  B2 x  x  x
2 x  B3 x  D3 D2
3 -  A3 B3 C3 x
4 x  x  x  x  x
```
此时，可以再逆走一遍，正向打印出路径即可。

# 标记已访问的节点

但是问题来了：现在程序在处理B3，那么它又怎么知道应该往B2、C3走而不是往A3走呢？若程序真的往A3走了，（即往回走了），那么程序在处理A3时又走向B3，之后又走向A3。结果陷入死循环，怎么办？

此时，就需要标记走过的节点。在处理完A3后，须在A3处打个标记，以防止后面“回流”到A3。同样，在处理完每个节点后，都要在上面打个标记。
以上就是宽搜的主要思路。可以提炼一下整个过程：
1. 将节点放入队列
2. 取出节点，并打上标记，以免重复访问
3. 分析该节点可以扩展出哪些节点，并将它们压入队列尾部。如果有必要，则可以在扩展节上附加一条信息：即这些扩展节点（子节点）都是由该节点（父节点）扩展而来
4. 重复以上步骤，直到访问到目标节点，此时就找到拉路径。若直到队列清空都没有访问到目标节点，则问题无解
# 宽搜的性质
可以看出，宽搜具有一下性质：
1. 找最短路径，宽搜找到的一定是由起始节点到目标节店最短的合法路径
2. 容易炸队列。在程序中，队列的长度不是无限的,而宽搜会把所有可以找到的节点全部扔到队列中。试想，若刚才的迷宫大小为$10000\times10000$，且大部分节点有2~3个岔路口，那么大概率队列会炸掉。
# 模板
## C++
```cpp
#include<bits/stdc++.h>
using namespace std;
void bfs(node start){ //宽度优先搜索
    queue<node> q; //队列
    q.push_back(start); //向队列中加入一个元素
    while(!q.empty()){ //队列非空
        node top = q.top(); //取出最前面的元素
        q.pop(); //取出后把元素从队列中删除
        do_sth(); //处理刚才的元素
        visited_flag(top); //标记刚才的元素访问过了，这样就不会重复访问了
        for(int i = 0;i < children(top);i++){ //对于刚才处理生成的所有衍生出来的元素
            if(!visited(child(top,i))){ //检查是否访问过
                q.push_back(child(top,i)); //如果还没有访问过那就将它加入队列中
            }
        }
    }
}
```

## Python
```py
def bfs(start): #宽度优先搜索
    q = [] #队列
    q.append(start) #向队列中加入一个元素
    while(len(q)): #队列非空
        top = q[0] #取出最前面的元素
        q.pop(0) #取出后把元素从队列中删除
        do_sth() #处理刚才的元素
        visited_flag(top) #标记刚才的元素访问过了，这样就不会重复访问了
        for i in range(children(top)): #对于刚才处理生成的所有衍生出来的元素
            if(not visited(child(top,i))): #检查是否访问过
                q.append(child(top,i)) #如果还没有访问过那就将它加入队列中
```
